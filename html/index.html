Willkommen

Auf dieser Seite notiere ich Dinge von mir für mich. Das betrifft meistens Themen rund um alte PCs, Linux oder Programmierung. Bisher hab ich das in Github gemacht, allerdings ist das nicht so schön wie eine eigene Webseite. Also hier ist sie.

Diese Seite besteht nur aus statischem HTML, CSS und ein bisschen clientseitigem Javascript. Es werden keine Frameworks oder Bibliotheken verwendet. Ein schlaues Backend gibt es leider auch nicht.

Updates erfolgen über ein GIT Repository. Nach einem Commit in den Main Branch veröffentlicht eine Github Action die Dateien auf dem Webserver. Bei Fehlern auf der Seite also gerne einen pull request erstellen. 😀
