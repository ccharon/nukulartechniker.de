Greaseweazle

A Greaseweazle is an open-source hardware + software project for reading and writing floppy disks. It connects via USB and can handle various floppy disk formats. The software is written in Python and can be used on different operating systems.


Installation under Linux (Gentoo) in a separate python venv

$ python -m venv $HOME/greaseweazle/venv
$ source $HOME/greaseweazle/venv/bin/activate
$ pip install git+https://github.com/keirf/greaseweazle@latest

To use the now installed gw the venv has to be active. To activate it later use:

$ source $HOME/greaseweazle/bin/activate

Or use a little script, save it as: $HOME/bin/weazle.sh. This enters a new shell with the venv active and already in a directory to store images.

#!/bin/env bash

[ ! -d $HOME/greaseweazle/images ] && mkdir -p $HOME/greaseweazle/images
cd $HOME/greaseweazle/images

bash --init-file <(echo "source $HOME/greaseweazle/venv/bin/activate ; gw reset ; gw info")

Testing if the greaseweazle is recognized

Attach the device via USB. For me it shows up as /dev/ttyACM0

$ gw info
